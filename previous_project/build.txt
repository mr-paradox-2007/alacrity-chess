## Manual Build Instructions

### Prerequisites
- C++17 compiler (g++)
- Standard development libraries
- Data structures compiled and headers in place

### Compilation Command
```bash
g++ -std=c++17 -pthread \
    -I./includes \
    -I/usr/include \
    src/server/network/server.cpp \
    src/server/api/auth_api.cpp \
    src/server/main.cpp \
    -o bin/alacrity_chess_server
```

### Linking Libraries
- libpthread (for threading)
- Standard C++ library

### Expected Output
A binary executable at `bin/alacrity_chess_server` approximately 500-800 KB

### Execution
```bash
./bin/alacrity_chess_server
```

Server will:
1. Initialize all 5 data structures
2. Initialize Auth API
3. Start listener thread on port 9999
4. Start worker thread for message processing
5. Listen for incoming TCP connections
