IF ANYTHING BREAKS
==================

Port 8080 in use:
  pkill -9 -f chess_server
  sleep 2
  ./server/bin/chess_server

Frontend shows "Route not found":
  - Verify server: curl http://localhost:8080/health
  - Check path: ls client/index.html
  - Restart server from correct directory

API returns "Missing token":
  - Get token from /auth/register or /auth/login
  - Include in header: Authorization: Bearer TOKEN

Server won't compile:
  cd server
  g++ -std=c++17 -pthread -O2 main.cpp -o bin/chess_server
